# ChatLLM

## Overview

A multimodal AI chat application leveraging Google's Gemini models for seamless text and image analysis, featuring a responsive, polished dark-mode UI. Built with a modern stack including Next.js 15, TypeScript, and Tailwind CSS, it implements secure client-side API key management and modular architecture.

## Why i built this

To explore the multimodal capabilities of Google's Gemini models by creating a seamless interface that unifies text, image, and voice interactions.
